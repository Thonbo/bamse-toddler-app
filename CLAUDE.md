# CLAUDE.md — preferences for this repo

## Git workflow
- **Commit directly to `main`.** Branch previews aren't reviewable
  here; only the live Netlify deploy from `main` is. Every change goes
  to `main` from the start — don't create or push feature branches.
- **Never ask about creating pull requests.** The user merges feature
  branches straight to `main` themselves. Don't suggest opening a PR
  after a push; just report the push and stop.
