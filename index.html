mapPoint (feOffset)

SVG/JS has no mapPoint property on the displacement filter — this reproduces it by translating the displacement map via feOffset. Equivalent to AS3 DisplacementMapFilter.mapPoint. Pairs well with Stitch tiles to avoid edge artifacts.

Gyroscope drive (mapPoint ← device tilt)

Drives mapPoint directly from device orientation: gamma (left-right tilt) → x, beta (front-back tilt) → y. Overrides the constant drift while enabled. Tap Calibrate to make the current pose the zero-point.
