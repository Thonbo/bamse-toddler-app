# Bamse & Kylling Toddler App

A locked-down toddler-safe web app with looping Bamse & Kylling videos and a tap game.

## Features

- 🎵 Jodlesangen, 🚗 Brum Brum, 🎺 Okay Okay, 🐻 Bamses Sang — all looping
- 🎮 Tap game with bubbles, confetti, fireworks, and emoji pops
- 🔒 Long-press top-right corner for 2 seconds to exit (toddler-proof)
- 🖥️ Auto fullscreen + landscape lock + screen wake lock
- 🔇 No external assets — sounds generated via Web Audio

## Use on Android

1. Open the deployed URL in Chrome
2. Menu → "Add to Home Screen"
3. Pair with Android Screen Pinning (Settings → Security → Screen Pinning) for kiosk-mode lockdown

## Deploy

This is a single static `index.html` — drop it on any host. Netlify auto-deploys on push.
